Meus Projetos

Site Institucional para a "EcoSustain"

Desenvolvimento de um site institucional completo para a EcoSustain, uma startup focada em soluções de sustentabilidade. O projeto incluiu a criação de um layout moderno e responsivo, a implementação de um sistema de gerenciamento de conteúdo (CMS) para o blog da empresa e a otimização para motores de busca (SEO) para aumentar a visibilidade online.

Tecnologias utilizadas: HTML5, CSS3, JavaScript, e um CMS headless.

Gestor de Tarefas Pro

Uma aplicação web interativa para gerenciamento de tarefas (To-Do List) construída com React. Permite aos usuários adicionar, editar, remover e marcar tarefas como concluídas. A aplicação utiliza o localStorage do navegador para persistir os dados, garantindo que as tarefas não sejam perdidas ao recarregar a página.

Tecnologias utilizadas: React, CSS Modules, JavaScript (ES6+).

Landing Page para App "Connecta"

Criação de uma landing page responsiva e visualmente atraente para o lançamento do aplicativo móvel "Connecta". O objetivo da página era apresentar as funcionalidades do app, incentivar o pré-cadastro de usuários e capturar leads. O design foi focado na experiência do usuário e na conversão.

Tecnologias utilizadas: HTML5, CSS3 (com Flexbox e Grid), JavaScript.
